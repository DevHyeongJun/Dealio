# Dealio
## 계약 관리 수정 사항
0. 계약 관리 입력 수정은 견적서 관리와 마찬가지로 별도 페이지가 존재해야 한다.
 - 누가 작성했는지 누가 수정했는지 변경 이력이 필요하다.
1. 계약 관리에 거래처명은 직접 입력이 아니고 고객 관리와 연결된 정보로 입력되야 한다. 현재 고객 마스터 연결을 거래처라고 생각하면 된다.
2. 계약금액은 입력 시 인풋박스 하단에 금일원정 형식으로 한글로 치환되서 보여주고, 금액 입력 시 알아서 #,###,### 포멧으로 치환해라.
3. 정산금액도 마찬가지다.
4. 계약 관리 페이지 검색 필터 바로 위에는 현재 계약 현황을 한 눈에 볼 수 있게 금액 들을 표출해줘라.
 - 정산된 금액 / 미정산된 금액 등 전체적인 금액을 한 눈에 볼 수 있어야 됨.

 ## 시스템 설정
1. 견적서 메일 내용 템플릿도 직접 수정하거나 미리보기 볼 수 있게 해라. 템플릿 수정 버튼 누르면 팝업창으로 표출된 상태에서 수정하는거다.
